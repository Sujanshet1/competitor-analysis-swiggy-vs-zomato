# 🍔 Competitor Analysis – Swiggy vs Zomato

## Project Overview

This project compares two leading Indian food delivery platforms—Swiggy and Zomato—to understand their strengths, weaknesses, and opportunities for improvement.

## Objectives

- Compare product features
- Analyze customer experience
- Identify competitive advantages
- Recommend product improvements

## Competitors

- Swiggy
- Zomato

## Evaluation Criteria

- Delivery Speed
- Pricing
- User Interface
- Restaurant Variety
- Customer Support
- Live Order Tracking
- Loyalty Program
- Offers & Discounts

## Tools Used

- GitHub
- Google Sheets
- Markdown
- AI-assisted Analysis
